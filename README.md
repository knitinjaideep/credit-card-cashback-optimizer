# Credit Card Cashback Optimizer

A React-based web application that helps users optimize their credit card usage by analyzing spending patterns and recommending the best credit card combinations for maximum cashback rewards.

## Features

- Interactive dashboard with spending analysis
- Credit card comparison and selection
- Real-time cashback calculations
- Visual data representation with charts
- Dark/Light theme support
- Responsive design

## Tech Stack

- React
- TypeScript
- Chart.js for data visualization
- Vite for build tooling
- CSS3 with modern features

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/credit-card-cashback-optimizer.git
cd credit-card-cashback-optimizer
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Usage

1. Select credit cards from the available options
2. View recommendations based on your spending patterns
3. Analyze cashback potential through interactive charts
4. Compare different card combinations
5. Toggle between light and dark themes
